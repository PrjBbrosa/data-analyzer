Precision Light

推荐作为默认方向。它最适合当前工具：高可读、低学习成本、适合长时间处理通道和图表。

主操作蓝 面板底色 主文字 辅助强调

落地优先级

1统一 QSS token、字体、边框、按钮、输入框、滚动条。
2替换 toolbar 和 pane 内 emoji，改成统一线性图标。
3升级左侧文件 row、通道颜色 swatch、图表颜色同步。
4重做 ChartStack 工作区、cursor pill、stats strip 和 Inspector section。

选择建议

  • 默认产品：选 Precision Light。
  • 夜间/演示：可提供 Graphite Lab 作为后续主题。
  • 想要更现代：Aero Glass 适合展示，但透明度要克制。
  • 生产/标定环境：Signal Studio 更耐看，视觉疲劳低。
