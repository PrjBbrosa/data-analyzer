坐标轴设置 · 三个可选方向

目标是把控件变轻：数字框去掉上下微调箭头，下拉框保留箭头；FFT Time 和 Order 的 X/Y/色阶只显示属于当前图的含义；密集数字收起来或分层；Order 图和设置区去掉灰底。

数字框：无上下箭头 下拉框：保留下拉箭头 Order：白底
推荐 A

自动摘要 + 点“编辑”再展开

最适合现在的问题：默认只看轴名和自动范围，手动时再展开 min/max，数字不会堆在一行。

优点：最少改动、最清晰。把不常改的 min/max 收到“编辑”里，能直接解决截图里数字挤成一团的问题。
方案 B

当前 Inspector 内，只显示当前图的轴

不新增外层面板。每个 Inspector 仍然独立，只把当前图的 X/Y/色阶语义、自动/手动范围显示清楚。

方案 C

默认只给摘要，高级设置放抽屉

界面最干净，适合多数时候只调色图/单位；完整 min/max 都进“高级坐标范围”。
